# Claude Code マスタリー実習テンプレート

このテンプレートは [Claude Code Mastery](https://claude-mastery-app-three.vercel.app) サイトと併せて使う、**実習用のプロンプト集**です。
上から順に 1 つずつ Claude Code に投げていくことで、サイトの全内容を「読むだけ」ではなく「実際に動かして」学べるようになっています。

## 使い方

1. このファイルをダウンロードし、新しい空のフォルダに置きます。
2. そのフォルダで `claude` を起動します。
3. サイトで項目を読んだあと、対応するプロンプトをコピーして Claude に貼り付けます。
4. 結果を見ながら、必要に応じて追加質問をして理解を深めます。

> 注意：専門用語には「保管場所（リポジトリ）」のような注釈を付けています。わからない言葉が出てきたら、そのまま Claude に「これって何？」と聞いて大丈夫です。

## 🪝 Hooks

### 12. PostToolUse Hook（自動フォーマット）
```
.claude/settings.json に PostToolUse フックを追加してください。
内容は「Write または Edit のあとに npm run lint を走らせる」です。
既存の設定は壊さないでください。
```

### 13. PreToolUse Hook（ファイルサイズガード）
```
800 行を超えるファイルを書こうとしたときに警告する PreToolUse フックを追加してください。
シェルスクリプトは scripts/hooks/ に保存してください。
```

### 14. Stop Hook（最終ビルド確認）
```
セッション終了時に npm run build を実行して、
ビルドが通らなければ警告する Stop フックを追加してください。
```

## 🤖 Subagents & Agent ツール

### 15. Explore agent
```
Explore agent を使って、このプロジェクトの全体構造を調べてください。
主要なフォルダ・ファイル・依存関係を 200 字以内でまとめてください。
```

### 16. Planner agent
```
「ユーザー認証機能（メール＋パスワード）を追加する」というタスクを、
Planner agent に渡して実装計画を作らせてください。

```

### 17. Code Reviewer agent
```
直近のコミットで変更されたファイルを Code Reviewer agent にレビューさせてください。
CRITICAL / HIGH / MEDIUM の 3 段階で指摘を分類してください。
```

### 18. 並列 agent 実行
```
3 つの agent を同時に起動してください：
1) Explore で全体構造調査
2) security-reviewer で脆弱性チェック
3) refactor-cleaner で不要コード検出
結果を 1 つのレポートにまとめてください。
```

### 19. Background agent
```
「全ファイルの typo を探して一覧化する」タスクを Background agent で走らせてください。
完了するまで他の作業を進められるようにしてください。
```
